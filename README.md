# My Unity Game

Tending Card Game

## Setup (developer machine)

1. Install Unity (the version defined in `ProjectSettings/ProjectVersion.txt`).
2. In Unity Editor: Edit → Project Settings → Editor:
   - Version Control Mode: Visible Meta Files
   - Asset Serialization: Force Text
3. Install Git and Git LFS:
   - https://git-scm.com/
   - https://git-lfs.github.com/
